# Formatos bicolumnares — EduMobility

Esta carpeta contiene los formatos bicolumnares asociados a los casos de uso del proyecto EduMobility.

Los formatos se organizan por subsistema funcional, de acuerdo con la división de requerimientos del proyecto:

- RF1 — Catálogo de Electivas
- RF2 — Retroalimentación Estudiantil
- RF3 — Gestión de Preaprobaciones
- RF4 — Agente Inteligente de Consulta
- RF5 — Integración Institucional y Autenticación

Cada subcarpeta reúne los formatos bicolumnares correspondientes a los casos de uso de su respectivo subsistema.

Los formatos se elaboran por caso de uso, no por cada requerimiento funcional atómico. Cuando varios requerimientos pertenecen al mismo flujo de interacción, se agrupan en un solo formato bicolumnar.
