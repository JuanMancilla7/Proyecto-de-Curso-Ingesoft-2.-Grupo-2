# RF3 — Gestión de Preaprobaciones

Esta carpeta contiene los formatos bicolumnares correspondientes a los casos de uso del subsistema RF3 — Gestión de Preaprobaciones de Equivalencias.

Los formatos de este subsistema se elaboran con base en los requerimientos funcionales actualizados, los casos de uso definidos por el equipo y la corrección del flujo sin asistente académica.
